### Installation Steps

#### 1. Update System Packages

```bash copy
sudo apt-get update
```

#### 2. Install Git

```bash copy
sudo apt-get install git
```

#### 3. Install Python

```bash copy
sudo apt-get install python3-dev python3.10-dev python3-setuptools python3-pip python3-distutils
```

#### 4. Install MariaDB

```bash copy
sudo apt install mariadb-server mariadb-client
```

Use `SPACE`, `ARROW`, and `ENTER` keys to handle prompts.

#### 5. Install Redis

```bash copy
sudo apt-get install redis-server
```

#### 6. Install Dependencies

```bash copy
sudo apt-get install xvfb libfontconfig wkhtmltopdf libmysqlclient-dev
```

#### 7. Secure MySQL

```bash copy
sudo mysql_secure_installation
```

Enter details as follows:

- Switch to unix_socket auth: `Y`
- Change root password: `Y`
- Disallow remote root login: `N`
- Remove anonymous/test DB: `Y`

#### 8. Update MySQL Config

```bash copy
sudo nano /etc/mysql/my.cnf
```

Paste:

```
[mysqld]
character-set-client-handshake = FALSE
character-set-server = utf8mb4
collation-server = utf8mb4_unicode_ci

[mysql]
default-character-set = utf8mb4
```

Restart:

```bash copy
sudo service mysql restart
```

#### 9. Install Node, NPM, Yarn

```bash copy
sudo apt install curl
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
source ~/.profile
nvm install 16.15.0
sudo apt-get install npm
sudo npm install -g yarn
```

#### 10. Create ERP Folder

```bash copy
mkdir erp && cd erp
```

#### 11. Install Frappe Bench

```bash copy
sudo pip3 install frappe-bench
```

#### 12. Initialize Bench

```bash copy
bench init --frappe-branch version-14 frappe-bench
```

#### 13. Enter Bench Directory

```bash copy
cd frappe-bench
```

#### 14. Create a New Site

```bash copy
bench new-site yoursite.local
```

#### 15. Install ERPNext App

```bash copy
bench get-app --branch version-14 erpnext
bench --site yoursite.local install-app erpnext
```

### âœ… ERPNext is now installed!

### Access via Windows Browser

#### 16. Enable SSH & Port Access

```bash copy
sudo apt install openssh-server
sudo ufw enable
sudo ufw allow 8000
sudo ufw allow 3306
```

#### 17. Setup Port Forwarding in VirtualBox

- Go to **Settings â†’ Network â†’ Advanced â†’ Port Forwarding**
- Add:
  - Host Port: `8000`, Guest Port: `8000`
  - Host Port: `3306`, Guest Port: `3306`

#### 18. Run Frappe Server

```bash copy
bench use yoursite.local
bench start
```

Open your browser and go to [http://localhost:8000](http://localhost:8000).

- **Username:** `Administrator`
- **Password:** The one you set during site creation
